# Section 02 Goal: Airline App Deployment

## Goal
Deploy the airline workload into Kubernetes and teach how to inspect and interact with it using kubectl.

## Scope
- Deploy the airline services
- Explain namespaces, pods, services, configmaps, and PVCs in context
- Show common kubectl commands for inspection and troubleshooting

## Out of scope
- Cost/FinOps analysis
- AI recommendations
- Issue tracker integration

## Success criteria
The learner can deploy the app, verify the resources, and navigate the cluster with kubectl.
