# Section 01 Goal: Cluster Foundation

## Goal
Set up the Kubernetes learning environment: cluster access, core namespaces, and the minimum cluster baseline needed for the rest of the flow.

## Scope
- Create or connect to the K8 cluster
- Validate kubectl access
- Create the namespaces used later in the story
- Establish the baseline environment only

## Out of scope
- Application deployment
- Cost analysis
- AI/Bedrock logic
- Issue creation

## Success criteria
A learner can reach the cluster and clearly see the namespaces that will be used in later sections.
