# Section 03 Guide: FinOps Problems

## Teach
Show why the cluster is difficult to manage at scale: missing tags, orphaned resources, and unclear ownership.

## Use
- orphaned resource manifests
- tagging rule examples
- `kubectl` inspection from section 02

## Folder intent
- `manifests/`: bad or orphaned resources used to demonstrate the problem
- `examples/`: tagging rules, before/after snapshots, and evidence notes

## Keep the focus
This section frames the problem only. The agent solution starts in section 04.
