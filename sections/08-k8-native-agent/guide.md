# Section 08 Guide: Kubernetes-Native Agent

## Teach
Package the agent as a container, deploy it in Kubernetes, and run it on a schedule.

## Use
- container build/deploy steps
- namespace separation
- CronJob scheduling

## Folder intent
- `docker/`: build and image notes
- `manifests/`: Kubernetes deployment manifests
- `cronjob/`: scheduled execution resources

## Keep the focus
This section assumes the full local flow already works.
